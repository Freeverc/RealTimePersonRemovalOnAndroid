# Real Time Person Removal
A real time person removal tool on android based on unity.

## Relocalization
- EasyAR, orb based

## Image Retrival
- Pose based

## Person Detection
- MobileNet + SSD

## Image Fusion
- Poission fusion, linar fusion

## Real Time
- Tracking, optical flow  

## UI
- Unity
